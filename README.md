# English Education Web App

This monorepo houses the React frontend and Express backend for the IELTS teaching platform.

## Getting Started

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm run prisma:generate
npm run dev
```
